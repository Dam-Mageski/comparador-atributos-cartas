## Comparação com dois atributos (menus dinâmicos)

- Escolha 2 atributos diferentes via menus (`switch`).
- Regras:
  - Geral: maior vence
  - **Exceção:** Densidade Demográfica → **menor vence**
- Soma os dois atributos; maior soma vence; se igual → **Empate!**

### Compilar e executar
```bash
gcc comparador_atributos.c -o comparador
./comparador
